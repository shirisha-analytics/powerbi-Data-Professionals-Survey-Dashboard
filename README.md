## powerbi-Data-Professionals-Survey-Dashboard
# Dashboard Features
-- Average Salary by Job Title – Compare earnings across different roles
-- Favorite Programming Languages – Identify popular tools in the industry
-- Survey Respondents by Country – Geographic distribution of participants
-- Average Salary by Gender – Gender-based salary comparison
-- Work-Life Happiness – Insights into job satisfaction
-- Current Salary Overview – Distribution of salaries across respondents

# Tools & Technologies
- Microsoft Power BI Desktop – Data visualization and dashboard creation
- Excel / CSV – Data source
- GitHub – Project hosting
